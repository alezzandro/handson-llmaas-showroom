== Docling

In this exercise we will use *Docling Serve* to convert documents to a format that is more convenient during a tokenization process like Markdown.

=== Creating the Docling application

Follow these simple steps to create the *Docling Web Client* application

* Click on `Create a new application`
* Select `Docling for Parasol AI Studio`
* Enter `docling` as the name
* Hit `Create Application`

The application is shown as **starting** now

[.bordershadow]
image::03/04-docling_starting.png[width="100%"]

A few moments later an **open** link will appear

[.bordershadow]
image::03/04-docling_open.png[width="100%"]

=== Processing documents with the Docling Web Client

In the **Docling Web Client** under **Input Sources** enter the following URL:

[source,bash,role=execute,subs=attributes+]
----
https://www.redhat.com/en
----

The click on **Process URL(s)** leaving all options at default, this will convert the HTML site to Markdown.

[.bordershadow]
image::03/04-docling_process.png[width="100%"]

Now experiment with other document input and output formats, you can also download the results.
